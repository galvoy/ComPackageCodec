
# ComPackageCodec

## 特殊字节转义规则

特殊字节：

1. 255：包头
2. 254：包尾

转义方法1：

转义符253

| 原始字节 | 转义编码 |
| -------- | -------- |
| 253      | 253 0    |
| 254      | 253 1    |
| 255      | 253 2    |

转义方法2：

base64编码

## 包格式

| 字段类型 | 大小（byte） | 说明               |
| -------- | ------------ | ------------------ |
| Head     | 1            | 255                |
| Version  | 1            |                    |
| Length   | 2            | 高八位，低八位     |
| Content  | Length       |                    |
| CRC      | 2            | 校验目标为 Content |
| End      | 1            | 254                |
